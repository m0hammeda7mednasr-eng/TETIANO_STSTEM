WHERE email = 'testadmin@example.com';

## ❌ لو حاجة مش ظاهرة؟

### المشكلة: مش شايف ميزات المدير

**الحل:**

1. تأكد إنك عملت UPDATE في Supabase
2. سجل خروج من التطبيق
3. سجل دخول تاني
4. افتح Developer Console (F12)
5. اكتب: `localStorage.getItem('user')`
6. لازم تشوف: `"role":"admin"`

### المشكلة: صفحة Users فاضية

**الحل:**

1. افتح Developer Console (F12)
2. اذهب إلى Network tab
3. حدث الصفحة
4. شوف الـ request لـ `/api/users`
5. لو 403: المستخدم مش admin
6. لو 401: مفيش token

### المشكلة: Backend مش شغال

**الحل:**

```powershell
# أوقف أي process على port 5000
Get-NetTCPConnection -LocalPort 5000 | Select-Object -ExpandProperty OwningProcess | ForEach-Object { Stop-Process -Id $_ -Force }

# شغل Backend تاني
cd backend
npm run dev
```
