## 🎯 طرق التطبيق

### 1. لكل وحدة (Per Unit)

```
الوصف: التكلفة تتضرب في عدد الوحدات المباعة
مثال:
  إعلانات: $5 لكل وحدة
  بعت 10 وحدات
  التكلفة الإجمالية: $5 × 10 = $50
```

### 2. لكل طلب (Per Order)

```
الوصف: تكلفة ثابتة لكل طلب
مثال:
  شحن: $10 لكل طلب
  بعت 10 وحدات في طلب واحد
  التكلفة الإجمالية: $10
```

### 3. ثابت (Fixed)

```
الوصف: تكلفة ثابتة بغض النظر عن الكمية
مثال:
  إيجار المخزن: $100 شهرياً
  التكلفة الإجمالية: $100
```

## 🧮 معادلات الحساب

### صافي الربح

```
صافي الربح = الإيرادات - التكلفة - التكاليف التشغيلية
```

### هامش الربح

```
هامش الربح = (صافي الربح ÷ الإيرادات) × 100
```

### مثال عملي كامل

```
المنتج: iPhone Case
─────────────────────────
سعر البيع:              $100
سعر التكلفة:            -$40
                        ─────
الربح الإجمالي:          $60

التكاليف التشغيلية:
  - إعلانات فيسبوك:     -$10
  - شحن:                -$5
  - تغليف:              -$2
                        ─────
إجمالي التشغيلية:       -$17

═════════════════════════
صافي الربح:              $43
هامش الربح:              43%
```

## 🎨 الألوان والمؤشرات

### ألوان البطاقات الإحصائية

- 🔵 **أزرق**: إجمالي الإيرادات
- 🟠 **برتقالي**: تكلفة المنتجات
- 🟡 **أصفر**: التكاليف التشغيلية
- 🟢 **أخضر**: صافي الربح
- 🟣 **بنفسجي**: متوسط هامش الربح

### ألوان هامش الربح

- 🟢 **أخضر**: ربح عالي (> 30%)
- 🔵 **أزرق**: ربح متوسط (15-30%)
- 🟡 **أصفر**: ربح منخفض (0-15%)
- ⚪ **رمادي**: بدون ربح أو خسارة

### حالة الطلب

- ✅ **طلب مربح**: صافي الربح > 0
- ❌ **طلب خاسر**: صافي الربح < 0
- ⚪ **بدون ربح**: صافي الربح = 0
