#### الطريقة الثانية: SSH Key

```bash
# إنشاء SSH Key
ssh-keygen -t rsa -b 4096 -C "your-email@example.com"

# إضافة SSH Key لـ GitHub
# ثم تغيير remote URL
git remote set-url origin git@github.com:m0hammeda7mednasr-eng/TETIANO_STSTEM.git
git push -u origin main
```
