# ✅ إصلاح المهام وسعر التكلفة

## المشاكل المحلولة

### 1. مشكلة إنشاء المهام ❌ → ✅

**المشكلة:**

- عند محاولة إنشاء مهمة جديدة، كان يظهر خطأ "فشل إنشاء المهمة"

**السبب:**

- ملف `Tasks.jsx` كان يستخدم `axios` مباشرة بدلاً من `api` utility
- كان يضيف `/api` في الـ URL يدويًا، مما يسبب URL خاطئ: `/api/api/tasks`
- الـ token كان يُضاف يدويًا بدلاً من استخدام الـ interceptor

**الحل:**

- تم استبدال كل استخدامات `axios` بـ `api` من `../utils/api`
- إزالة `/api` من الـ URLs (لأن `api` utility يضيفها تلقائيًا)
- إزالة إضافة الـ token يدويًا (لأن الـ interceptor يضيفه تلقائيًا)

**الملفات المعدلة:**

- ✅ `frontend/src/pages/Tasks.jsx`

### 2. سحب سعر التكلفة من شوبيفاي ❌ → ✅

**المشكلة:**

- سعر التكلفة (Cost Price) لم يكن يُسحب من شوبيفاي تلقائيًا
- كان يجب إدخاله يدويًا

**الحل:**

- تم تحديث `ShopifyService.getProductsFromShopify()` لاستخراج `cost` من الـ variant
- شوبيفاي يخزن سعر التكلفة في `variant.cost` أو `variant.cost_price`
- الآن عند المزامنة، سعر التكلفة يُسحب تلقائيًا ويُحفظ في قاعدة البيانات

**الملفات المعدلة:**

- ✅ `backend/src/services/shopifyService.js`
- ✅ `backend/src/services/productManagementService.js`

## التفاصيل التقنية

### إصلاح Tasks.jsx

**قبل:**

```javascript
import axios from "axios";

const fetchTasks = async () => {
  const token = localStorage.getItem("token");
  const response = await axios.get("/api/tasks", {
    headers: { Authorization: `Bearer ${token}` },
  });
  // ...
};
```

**بعد:**

```javascript
import api from "../utils/api";

const fetchTasks = async () => {
  const response = await api.get("/tasks");
  // Token يُضاف تلقائيًا من الـ interceptor
  // ...
};
```

### إصلاح Cost Price

**قبل:**

```javascript
return allProducts.map((product) => ({
  // ...
  price: product.variants[0]?.price || 0,
  // cost_price مفقود!
}));
```

**بعد:**

```javascript
return allProducts.map((product) => {
  const firstVariant = product.variants[0] || {};
  const costPrice = firstVariant.cost || firstVariant.cost_price || 0;

  return {
    // ...
    price: firstVariant.price || 0,
    cost_price: costPrice, // ✅ يُسحب من شوبيفاي
  };
});
```

## كيفية الاستخدام

### 1. إنشاء مهمة جديدة

1. افتح صفحة "إدارة المهام"
2. اضغط على "إضافة مهمة جديدة"
3. املأ البيانات:
   - عنوان المهمة
   - الوصف
   - تعيين إلى (اختر مستخدم)
   - الأولوية (منخفضة / متوسطة / عالية / عاجلة)
   - تاريخ الاستحقاق
4. اضغط "إضافة"
5. ✅ المهمة ستُنشأ بنجاح!

### 2. سعر التكلفة من شوبيفاي

1. تأكد أن سعر التكلفة موجود في شوبيفاي:
   - افتح المنتج في شوبيفاي
   - اذهب إلى Variant
   - تأكد من وجود "Cost per item"
2. قم بالمزامنة:
   - اذهب إلى "الإعدادات"
   - اضغط "مزامنة البيانات"
3. ✅ سعر التكلفة سيُسحب تلقائيًا!
4. افتح تفاصيل المنتج لرؤية سعر التكلفة

## ملاحظات مهمة

### سعر التكلفة في شوبيفاي

- شوبيفاي يخزن سعر التكلفة في `variant.cost`
- بعض الإصدارات القديمة تستخدم `variant.cost_price`
- الكود يدعم كلا الحالتين
- إذا لم يكن موجودًا في شوبيفاي، سيكون 0

### المهام

- فقط الـ Admin يمكنه إنشاء وحذف المهام
- المستخدمون العاديون يمكنهم رؤية مهامهم فقط
- الـ Admin يرى كل المهام
- يمكن تحديث حالة المهمة من القائمة مباشرة
