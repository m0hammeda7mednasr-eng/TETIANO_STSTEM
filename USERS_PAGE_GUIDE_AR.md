# 👥 دليل صفحة المستخدمين - للمدير

## 🎯 كيف تدخل على صفحة المستخدمين؟

### الطريقة 1: من Sidebar

```
1. شوف القائمة اليمين
2. اضغط على "المستخدمين"
3. هتفتح صفحة /users
```

### الطريقة 2: من Dashboard

```
1. اضغط زر "📋 إدارة المستخدمين" في الأعلى
2. أو اضغط على أي كارت (طلبات الصلاحيات أو التقارير)
3. هتفتح صفحة /users
```

## 1️⃣ تبويب المستخدمين

### ما ستجده:

- جدول بجميع المستخدمين
- زر "إضافة مستخدم جديد" (أخضر في الأعلى)
- لكل مستخدم:
  - الاسم
  - البريد الإلكتروني
  - الدور (مدير أو مستخدم)
  - الحالة (نشط أو غير نشط)
  - أزرار الإجراءات

### الإجراءات المتاحة:

#### ✅ إضافة مستخدم جديد

```
1. اضغط زر "إضافة مستخدم جديد" (أخضر)
2. املأ البيانات:
   - الاسم
   - البريد الإلكتروني
   - كلمة المرور
   - الدور (مستخدم أو مدير)
3. اختر الصلاحيات:
   - ✅ عرض لوحة التحكم
   - ✅ عرض المنتجات
   - ✅ تعديل المنتجات
   - ✅ عرض الطلبات
   - ✅ تعديل الطلبات
   - ✅ عرض العملاء
   - ✅ تعديل العملاء
   - ✅ إدارة المستخدمين
   - ✅ إدارة الإعدادات
   - ✅ عرض الأرباح
4. اضغط "حفظ"
```

#### ✏️ تعديل صلاحيات مستخدم

```
1. اضغط أيقونة Shield (🛡️) بجانب المستخدم
2. ستفتح نافذة تعديل الصلاحيات
3. غيّر الصلاحيات كما تريد
4. اضغط "حفظ التغييرات"
```

#### 🗑️ حذف مستخدم

```
1. اضغط أيقونة سلة المهملات (🗑️)
2. ستظهر رسالة تأكيد
3. اضغط "موافق" للحذف
```

**ملاحظة:** لا يمكن حذف المدراء!

---

## 2️⃣ تبويب طلبات الصلاحيات

### ما ستجده:

- كروت لجميع طلبات الصلاحيات
- كل كارت يحتوي على:
  - اسم المستخدم
  - البريد الإلكتروني
  - الصلاحية المطلوبة
  - السبب
  - الحالة (معلق، موافق عليه، مرفوض)
  - التاريخ

### الإجراءات المتاحة:

#### ✅ الموافقة على طلب

```
1. اضغط زر "موافقة" (أخضر)
2. ستفتح نافذة التفاصيل
3. أضف ملاحظات (اختياري)
4. اضغط "موافقة"
5. الصلاحية ستُضاف تلقائياً للمستخدم
```

#### ❌ رفض طلب

```
1. اضغط زر "رفض" (أحمر)
2. أضف ملاحظات (اختياري)
3. اضغط "رفض"
4. المستخدم سيرى الرفض مع الملاحظات
```

### حالات الطلبات:

- 🟡 **معلق (pending)** - في انتظار مراجعتك
- 🟢 **موافق عليه (approved)** - تمت الموافقة
- 🔴 **مرفوض (rejected)** - تم الرفض

---

## 3️⃣ تبويب التقارير اليومية

### ما ستجده:

- كروت لجميع التقارير اليومية من الموظفين
- كل كارت يحتوي على:
  - اسم الموظف
  - البريد الإلكتروني
  - عنوان التقرير
  - التاريخ
  - الحالة
  - عدد المرفقات

### الإجراءات المتاحة:

#### 👁️ عرض تفاصيل التقرير

```
1. اضغط زر "عرض التفاصيل" (أزرق)
2. ستفتح نافذة بالتفاصيل الكاملة:
   - اسم الموظف
   - التاريخ
   - العنوان
   - الوصف
   - المهام المنجزة
   - الملاحظات
   - المرفقات (صور، Excel، PDF، Word)
3. يمكنك تحميل المرفقات
4. اضغط "إغلاق" للخروج
```

### حالات التقارير:

- 📝 **مسودة (draft)** - لم يتم إرساله بعد
- ✅ **مُرسل (submitted)** - تم إرسال التقرير
- 👁️ **تمت المراجعة (reviewed)** - تمت مراجعته

## 🎯 سيناريوهات الاستخدام

### سيناريو 1: إضافة موظف جديد

```
1. اذهب لتبويب "المستخدمين"
2. اضغط "إضافة مستخدم جديد"
3. املأ البيانات:
   - الاسم: أحمد محمد
   - البريد: ahmed@example.com
   - كلمة المرور: ********
   - الدور: مستخدم
4. اختر الصلاحيات:
   - ✅ عرض لوحة التحكم
   - ✅ عرض المنتجات
   - ✅ عرض الطلبات
   - ✅ عرض العملاء
5. اضغط "حفظ"
6. أرسل البيانات للموظف
```

### سيناريو 2: موظف طلب صلاحية جديدة

```
1. ستجد إشعار في Dashboard (كارت أصفر)
2. اذهب لصفحة المستخدمين
3. اذهب لتبويب "طلبات الصلاحيات"
4. ستجد الطلب الجديد (حالة: معلق)
5. اقرأ السبب
6. إذا موافق:
   - اضغط "موافقة"
   - أضف ملاحظات: "تمت الموافقة، يمكنك الآن تعديل المنتجات"
   - اضغط "موافقة"
7. إذا رافض:
   - اضغط "رفض"
   - أضف ملاحظات: "عذراً، هذه الصلاحية للمدراء فقط"
   - اضغط "رفض"
```

### سيناريو 3: مراجعة تقرير يومي

```
1. ستجد إشعار في Dashboard (كارت أزرق)
2. اذهب لصفحة المستخدمين
3. اذهب لتبويب "التقارير اليومية"
4. ستجد التقرير الجديد
5. اضغط "عرض التفاصيل"
6. اقرأ:
   - العنوان: "تقرير يوم 8 مارس 2026"
   - الوصف: "تم إنجاز المهام التالية..."
   - المهام المنجزة: "1. تحديث المنتجات..."
   - الملاحظات: "واجهت مشكلة في..."
7. شاهد المرفقات (إن وجدت)
8. حمّل الملفات إذا لزم الأمر
9. اضغط "إغلاق"
```

### سيناريو 4: تعديل صلاحيات موظف

```
1. اذهب لتبويب "المستخدمين"
2. ابحث عن الموظف
3. اضغط أيقونة Shield (🛡️)
4. ستفتح نافذة الصلاحيات
5. غيّر الصلاحيات:
   - ✅ تعديل المنتجات (كان ❌)
   - ✅ تعديل الطلبات (كان ❌)
6. اضغط "حفظ التغييرات"
7. الموظف سيرى الصفحات الجديدة في Sidebar تلقائياً
```

## 🔍 نصائح مهمة

### ✅ افعل:

- راجع طلبات الصلاحيات بانتظام
- اقرأ التقارير اليومية لمتابعة الموظفين
- أضف ملاحظات عند الموافقة/الرفض
- امنح الصلاحيات حسب الحاجة فقط

### ❌ لا تفعل:

- لا تمنح صلاحية "إدارة المستخدمين" لغير المدراء
- لا تحذف مستخدم لديه بيانات مهمة
- لا تشارك كلمات المرور
- لا تمنح جميع الصلاحيات لجميع الموظفين
