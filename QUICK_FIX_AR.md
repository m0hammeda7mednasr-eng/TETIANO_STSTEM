# ⚡ الحل السريع - مشاكل حساب المدير

## 🎯 المشكلة

المدير يرى:

- ❌ "طلب صلاحية" (مش المفروض)
- ❌ "تقاريري اليومية" (مش المفروض)
- ❌ Dashboard مش بيعرض الطلبات والتقارير

---

## ✅ الحل (3 خطوات فقط!)

### الخطوة 1: تأكد من الدور في قاعدة البيانات

افتح Supabase SQL Editor ونفذ:

```sql
-- شوف دورك
SELECT email, role FROM users WHERE email = 'your-email@example.com';
```

**إذا كان `role = 'user'`، نفذ:**

```sql
-- حول لـ admin
UPDATE users SET role = 'admin' WHERE email = 'your-email@example.com';
```

---

### الخطوة 2: امسح الـ Cache

اضغط على لوحة المفاتيح:

```
Ctrl + Shift + R
```

أو:

```
Ctrl + Shift + Delete
→ اختر "Cached images and files"
→ Clear data
```

---

### الخطوة 3: Logout ثم Login

```
1. اضغط زر "تسجيل الخروج"
2. سجل دخول تاني
3. تحقق من Sidebar
```

```
✅ رسوم بيانية
```
