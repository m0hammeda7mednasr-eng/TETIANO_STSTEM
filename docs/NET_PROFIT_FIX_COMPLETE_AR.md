   - افتح ملف `ADD_OPERATIONAL_COSTS_TABLE.sql` من مجلد المشروع

   - يجب أن ترى رسالة "Success. No rows returned"

#### إذا كان عمود cost_price مفقوداً:

نفّذ هذا الأمر أيضاً في SQL Editor:

```sql

ADD COLUMN IF NOT EXISTS cost_price DECIMAL(10, 2) DEFAULT 0;
```

## 📊 الميزات المتاحة

### البطاقات الإحصائية (5 بطاقات):

1. **إجمالي الإيرادات** (أزرق) - مجموع أسعار جميع المنتجات
2. **تكلفة المنتجات** (برتقالي) - مجموع تكاليف جميع المنتجات
3. **التكاليف التشغيلية** (أصفر) - مجموع جميع التكاليف التشغيلية
4. **صافي الربح** (أخضر) - الربح بعد خصم جميع التكاليف
5. **متوسط هامش الربح** (بنفسجي) - متوسط نسبة الربح

### جدول المنتجات:

- عرض جميع المنتجات مع الصور
- سعر البيع لكل منتج
- سعر التكلفة (قابل للتعديل)
- التكاليف التشغيلية (قابلة للإضافة والحذف)
- صافي الربح (محسوب تلقائياً)
- هامش الربح (محسوب تلقائياً)

### البحث:

- ابحث عن أي منتج بالاسم أو ID
- النتائج تظهر فوراً أثناء الكتابة

### تعديل سعر التكلفة:

1. انقر على أيقونة التعديل (✏️)
2. أدخل السعر الجديد
3. انقر حفظ (✓)
4. الإحصائيات تتحدث تلقائياً

### إدارة التكاليف التشغيلية:

#### إضافة تكلفة:

1. انقر "إضافة تكلفة" لأي منتج
2. املأ النموذج:
   - اسم التكلفة (مثال: إعلانات فيسبوك)
   - نوع التكلفة (إعلانات، عمليات، شحن، تغليف، أخرى)
   - المبلغ
   - طريقة التطبيق (لكل وحدة، لكل طلب، ثابت)
   - وصف (اختياري)
3. احفظ
4. الإحصائيات تتحدث تلقائياً

#### حذف تكلفة:

1. انقر على أيقونة الحذف (🗑️) بجانب التكلفة
2. أكّد الحذف
3. الإحصائيات تتحدث تلقائياً

## 🎯 الحسابات

### لكل منتج:

```
صافي الربح = سعر البيع - سعر التكلفة - مجموع التكاليف التشغيلية
هامش الربح = (صافي الربح / سعر البيع) × 100
```

### الإجمالي:

```
إجمالي الإيرادات = مجموع أسعار جميع المنتجات
تكلفة المنتجات = مجموع تكاليف جميع المنتجات
التكاليف التشغيلية = مجموع جميع التكاليف التشغيلية النشطة
صافي الربح = إجمالي الإيرادات - تكلفة المنتجات - التكاليف التشغيلية
متوسط هامش الربح = متوسط هوامش الربح لجميع المنتجات
```
