## الخطوة 4: الاختبار

### اختبار 1: تسجيل الدخول كـ Admin

1. سجل دخول بحساب Admin
2. افتح **Dashboard** (`/dashboard`)
   - ✅ المفروض تشوف: الإحصائيات، Charts، Quick Actions
3. افتح **لوحة تحكم الأدمن** (`/admin`)
   - ✅ المفروض تشوف: Activity Log (سجل النشاطات)
4. افتح **إدارة المستخدمين** (`/users`)
   - ✅ المفروض تشوف: قائمة المستخدمين

### اختبار 2: تسجيل الدخول كـ Employee (لو عندك حساب)

1. سجل خروج من Admin
2. سجل دخول بحساب Employee
3. افتح **Dashboard** (`/dashboard`)
   - ✅ المفروض تشوف: رسالة ترحيب + 4 أزرار كبيرة فقط
   - ❌ مش المفروض تشوف: إحصائيات أو Charts
4. افتح **المنتجات** (`/products`)
   - ✅ المفروض تشوف: كل المنتجات (نفس اللي الأدمن بيشوفها)
5. افتح **الطلبات** (`/orders`)
   - ✅ المفروض تشوف: كل الطلبات (نفس اللي الأدمن بيشوفها)

---

## الخطوة 5: اختبار Activity Log

### عشان تشوف Activity Log شغال:

1. سجل دخول كـ **Admin**
2. افتح **إدارة المستخدمين** (`/users`)
3. عدل صلاحيات أي يوزر
4. افتح **لوحة تحكم الأدمن** (`/admin`)
5. ✅ المفروض تشوف التعديل اللي عملته في Activity Log

## المشاكل المحتملة وحلولها:

### مشكلة 1: الباك إند مش بيشتغل

```bash
# تأكد إنك في مجلد backend
cd backend

# شغل npm start
npm start
```

### مشكلة 2: الفرونت إند مش بيشتغل

```bash
# تأكد إنك في مجلد frontend
cd frontend

# امسح node_modules وأعد التثبيت
rm -rf node_modules
npm install
npm start
```

### مشكلة 3: Dashboard فاضي لليوزر العادي

- ده صح! اليوزر العادي المفروض يشوف رسالة ترحيب و4 أزرار بس
- الإحصائيات والـ Charts للأدمن بس

### مشكلة 4: Activity Log فاضي

- ده طبيعي لو مفيش نشاطات
- جرب تعمل أي تعديل (مثلاً: عدل صلاحيات يوزر)
- بعدين ارجع لـ `/admin` وهتلاقي التعديل ظهر

### مشكلة 5: 403 Forbidden على Dashboard

- معناها اليوزر مش Admin في الداتابيز
- نفذ الـ SQL من ملف `MAKE_ALL_USERS_ADMIN.sql` في Supabase
- امسح localStorage: `localStorage.clear()` في Console
- سجل دخول تاني
