# 🚀 ابدأ استخدام ميزة حساب الربح

## خطوة واحدة مهمة قبل البدء!

### 1️⃣ نفذ SQL Script في Supabase

**مهم جداً:** يجب تنفيذ هذا الأمر أولاً!

1. افتح Supabase Dashboard
2. اذهب إلى SQL Editor
3. افتح الملف `ADD_COST_PRICE_AND_PROFIT.sql`
4. انسخ كل المحتوى والصقه في SQL Editor
5. اضغط "Run" أو "Execute"

✅ بعد التنفيذ، ستظهر رسالة نجاح

---

## 2️⃣ أعد تشغيل Backend و Frontend

### Backend:

```bash
cd backend
npm start
```

### Frontend:

```bash
cd frontend
npm start
```

---

## 3️⃣ جرب الميزة!

### إضافة سعر التكلفة:

1. اذهب إلى صفحة **المنتجات**
2. اضغط **"تعديل"** على أي منتج
3. أدخل:
   - السعر (مثلاً: 100)
   - **سعر التكلفة** (مثلاً: 60)
   - المخزون
4. سيظهر حساب الربح تلقائياً:
   - الربح لكل وحدة: 40
   - هامش الربح: 40%
5. احفظ التغييرات

### عرض الربح:

**في صفحة المنتجات:**

- صندوق أخضر على كل منتج يعرض الربح وهامش الربح

**في صفحة تفاصيل المنتج:**

- قسم كامل يعرض:
  - الربح لكل وحدة
  - هامش الربح %
  - الربح المحتمل (إجمالي المخزون)

**في صفحة تفاصيل الطلب:**

- قسم أخضر يعرض:
  - التكلفة الإجمالية للطلب
  - الربح الإجمالي
  - هامش الربح %
