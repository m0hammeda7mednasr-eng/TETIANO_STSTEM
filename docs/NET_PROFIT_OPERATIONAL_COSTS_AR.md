## الميزات الجديدة

### 1. التكاليف التشغيلية

- **أنواع التكاليف:**
  - إعلانات (Ads)
  - عمليات (Operations)
  - شحن (Shipping)
  - تغليف (Packaging)
  - أخرى (Other)

- **طرق التطبيق:**
  - لكل وحدة (Per Unit): يتم ضرب التكلفة في عدد الوحدات المباعة
  - لكل طلب (Per Order): تكلفة ثابتة لكل طلب
  - ثابت (Fixed): تكلفة ثابتة بغض النظر عن الكمية

### 2. حساب صافي الربح

```
صافي الربح = الإيرادات - تكلفة المنتجات - التكاليف التشغيلية
```

**مثال:**

- إيرادات: $100
- تكلفة المنتجات: $40
- الربح الإجمالي: $60
- تكاليف تشغيلية (إعلانات): $15
- **صافي الربح: $45**
- **هامش الربح الصافي: 45%**

### 3. المرتجعات (Refunds)

- عرض جميع المرتجعات للطلب
- تفاصيل كل مرتجع:
  - التاريخ
  - المبلغ
  - المنتجات المرتجعة
  - السبب (إن وجد)
- إجمالي المبالغ المرتجعة

### 4. الإلغاءات (Cancellations)

- تاريخ الإلغاء
- سبب الإلغاء
- عرض واضح للطلبات الملغاة

## Database Schema

### جدول operational_costs

```sql
CREATE TABLE operational_costs (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  product_id UUID REFERENCES products(id),
  cost_name VARCHAR(255) NOT NULL,
  cost_type VARCHAR(50) NOT NULL,
  amount DECIMAL(10, 2) NOT NULL,
  apply_to VARCHAR(50) DEFAULT 'per_unit',
  description TEXT,
  is_active BOOLEAN DEFAULT true,
  created_at TIMESTAMP,
  updated_at TIMESTAMP
);
```
