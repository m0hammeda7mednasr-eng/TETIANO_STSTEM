
1. افتح Supabase Dashboard
2. اذهب إلى SQL Editor
3. نفذ:

```sql

SET role = 'admin'
WHERE email = 'testadmin@example.com';
```

## 📝 ملاحظات مهمة

1. **كل الكود موجود وصحيح** - لا يوجد أي مشكلة في الكود
2. **المشكلة الوحيدة:** المستخدم يجب أن يكون `role = 'admin'` في Database
3. **بعد تحويل المستخدم إلى admin:** يجب تسجيل الخروج والدخول مرة أخرى
4. **localStorage يحفظ role:** تأكد من أن localStorage.getItem('user') يحتوي على `"role":"admin"`
